Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. You can use getElementById("id") to select by ID, getElementsByClassName("class") to select by class, and getElementsByTagName("tag") to select by tag. You can also use querySelector("selector") to get the first matching element and querySelectorAll("selector") to get all matching elements.

Question-3: What is event delegation in the DOM?

Event delegation is a way to handle events using a parent element instead of adding event listeners to multiple child elements. When a child element is clicked, the event "bubbles up" to the parent, and the parent checks which child was clicked. This makes the code easier to manage, especially for dynamic elements.

Question-4: What is event bubbling in the DOM?

Event bubbling means when an event happens on a child element, it first runs on that element and then moves up to its parent, then the grandparent, and so on. It keeps going up the DOM tree until it reaches the top. This helps in handling events at a higher level instead of adding them to every small element.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement("tag"). To add it, use parent.appendChild(element) or parent.append(element). To remove an element, use element.remove() or parent.removeChild(element). This helps in dynamically changing the webpage content.
